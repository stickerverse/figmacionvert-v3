Testing Instructions:

  1. Load this page in a browser with the enhanced chrome extension
  2. Capture the page using the extension's capture functionality
  3. Import into Figma via the plugin to test grid conversion
  4. Verify conversions:
    • Simple grids → nested Auto Layout frames
    • Complex grids → absolute positioning or hybrid approach
    • Grid annotations appear in node names and plugin data
    • fr units converted to proper flex-grow values
    • minmax() functions simplified to computed values
  5. Check console logs for grid conversion details

Expected Results:

  • Tests 1, 2, 3, 7, 8, 9: Nested Auto Layout (good conversion)
  • Tests 4, 10: Absolute positioning (complex features)
  • Tests 5, 6: Hybrid approach (mixed complexity)
