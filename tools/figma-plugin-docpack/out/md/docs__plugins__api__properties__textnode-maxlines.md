On this page

The maximum number of lines a text node can reach before it truncates. Only applicable when [`textTruncation`](/docs/plugins/api/properties/TextNode-texttruncation/) is set to `"ENDING"`.

Supported on:

*   [TextNode](/docs/plugins/api/TextNode/)

## Signature[​](#signature "Direct link to Signature")

### [maxLines](/docs/plugins/api/properties/TextNode-maxlines/): number | null

## Remarks[​](#remarks "Direct link to Remarks")

The value must be >= 1. To disable truncation at a maximum number of lines, set to `null`.
