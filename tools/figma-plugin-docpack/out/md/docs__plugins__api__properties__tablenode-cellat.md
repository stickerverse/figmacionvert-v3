Returns the table cell node at a specific cell coordinate.

Supported on:

*   [TableNode](/docs/plugins/api/TableNode/)

## Signature[​](#signature "Direct link to Signature")

### cellAt(rowIndex: number, columnIndex: number): [TableCellNode](/docs/plugins/api/TableCellNode/)

## Parameters[​](#parameters "Direct link to Parameters")

### rowIndex[​](#rowindex "Direct link to rowIndex")

The index of the row. Must satisfy `0 <= rowIndex < numRows`.

### columnIndex[​](#columnindex "Direct link to columnIndex")

The index of the column. Must satisfy `0 <= columnInde < numColumns`.
